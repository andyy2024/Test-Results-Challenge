# Test Results Challenge

This app displays a list of software releases with summary statistics and charts. It lets users browse release information, add new releases, and view an overview of the latest results.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal in your browser.

## Desicion Notes

I tried to mimic the clean design of the dashboard I saw during the interview. I kept it simple with no vivid colors or flashy elements; just a clean white layout with gray and black outlines.
